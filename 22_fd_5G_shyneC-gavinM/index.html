Now if you right-click Tab 2 and inspect, try to activate Tab 2. How?
  1. Find where it says tabs-title in a <li> tag, and change that to tabs-title is-active.
  2. Find where it says false in the corresponding <a> tag, and change that to true.
    3. Which one is corresponding? All should be labeled with panel#. Make sure to change the right one!
  3. Underneath all of the tab rectangles section, there should be another <div> tag that has the class as tabs-content.
  4. Find the corresponding <div> inside. Change the class to tabs-panel is-active instead of tabs-panel. Change aria-hidden to false instead of true.
    5. What do you notice?
